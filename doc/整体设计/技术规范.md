# 前端技术规范

## 1. 组件开发规范

### 1.1 组件示例

#### 展示型组件示例
```vue
<!-- 展示型组件示例 -->
<template>
  <div class="project-card" :class="{ 'is-highlighted': isHighlighted }">
    <div class="project-card__header">
      <h3 class="project-card__title">{{ title }}</h3>
      <span class="project-card__status" :class="statusClass">
        {{ status }}
      </span>
    </div>
    <div class="project-card__content">
      <slot name="content"></slot>
    </div>
    <div class="project-card__footer">
      <slot name="actions"></slot>
    </div>
  </div>
</template>

<script setup lang="ts">
interface Props {
  title: string
  status: string
  isHighlighted?: boolean
}

const props = withDefaults(defineProps<Props>(), {
  isHighlighted: false
})

const statusClass = computed(() => `status--${props.status.toLowerCase()}`)
</script>
```

#### 容器型组件示例
```vue
<!-- 容器型组件示例 -->
<template>
  <div class="project-list">

    </div>
    <project-pagination
      v-model:current="currentPage"
      :total="total"
      @change="handlePageChange"
    />
  </div>
</template>
```

### 1.2 样式规范

#### Grid系统配置
```scss
// styles/grid-system.scss

// Grid变量配置
:root {
  // 栅格列数
  --grid-columns: 12;
  
  // 栅格间距
  --grid-gap: 16px;
  
  // 容器宽度
  --container-width: 1200px;
  
  // 响应式断点
  --breakpoint-sm: 576px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 992px;
  --breakpoint-xl: 1200px;
  --breakpoint-xxl: 1400px;
}

// Grid容器
.container {
  width: 100%;
  margin: 0 auto;
  padding: 0 var(--grid-gap);
  
  @media (min-width: var(--breakpoint-sm)) {
    max-width: 540px;
  }
  
  @media (min-width: var(--breakpoint-md)) {
    max-width: 720px;
  }
  
  @media (min-width: var(--breakpoint-lg)) {
    max-width: 960px;
  }
  
  @media (min-width: var(--breakpoint-xl)) {
    max-width: var(--container-width);
  }
}

// Grid行
.row {
  display: grid;
  grid-template-columns: repeat(var(--grid-columns), 1fr);
  gap: var(--grid-gap);
}
```

#### Flex布局规范
```scss
// styles/flex-utils.scss

// Flex容器类
.flex {
  display: flex;
  
  &-inline {
    display: inline-flex;
  }
  
  // 主轴方向
  &-row {
    flex-direction: row;
  }
  
  &-column {
    flex-direction: column;
  }
  
  // 主轴对齐
  &-start {
    justify-content: flex-start;
  }
  
  &-center {
    justify-content: center;
  }
  
  &-end {
    justify-content: flex-end;
  }
  
  &-between {
    justify-content: space-between;
  }
  
  // 交叉轴对齐
  &-items {
    &-start {
      align-items: flex-start;
    }
    
    &-center {
      align-items: center;
    }
    
    &-end {
      align-items: flex-end;
    }
  }
  
  // 换行
  &-wrap {
    flex-wrap: wrap;
  }
  
  &-nowrap {
    flex-wrap: nowrap;
  }
}

// Flex子项类
.flex-item {
  // 弹性增长
  &-grow {
    flex-grow: 1;
  }
  
  // 弹性收缩
  &-shrink {
    flex-shrink: 1;
  }
  
  // 固定宽度
  @for $i from 1 through 12 {
    &-#{$i} {
      flex: 0 0 percentage($i / 12);
    }
  }
}
```
